## About Me

I’m **Sardar Abdalla**, a **C# / .NET Developer** with **4+ years** of experience delivering business-critical software in the UK.  
I focus on building **scalable .NET systems** with **Clean Architecture**, with strong attention to:

- **Maintainability**: clear layering, testable services, and predictable deployments  
- **Performance**: efficient APIs, data access, and UI flows that feel instant  
- **Security**: careful auth boundaries, validation, and safe integration patterns  
- **Real-world delivery**: shipping ERP/CRM/POS solutions that reduce operational time and increase reliability
